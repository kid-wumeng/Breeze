### padStart(value, len, char)

<params>
   <item>
      <name>value</name>
      <desc>will convert to string.</desc>
   </item>
   <item>
      <name>len</name>
      <type>number</type>
      <desc>optional, default is *10*, expected overall-length.</desc>
   </item>
   <item>
      <name>char</name>
      <type>string</type>
      <desc>the monocase used to padding.</desc>
   </item>
</params>


<example>

* car
